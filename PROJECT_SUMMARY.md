# Итог проекта - DTS to FLAC Converter

## Обзор

Проект был очищен и оптимизирован для создания эффективного решения конвертации аудиодорожек DTS-HD MA в FLAC с оптимизированным потреблением ресурсов.

## Финальная архитектура

### Выбранный стек технологий:
- **Backend**: Go (Golang) - для минимального потребления ресурсов
- **Frontend**: Vanilla JavaScript - легкий интерфейс
- **База данных**: SQLite - не требует отдельного сервера
- **Контейнеризация**: Docker + Nginx

### Ключевые преимущества:
- **Потребление памяти**: 10-20MB (против 50-100MB у Node.js)
- **Производительность**: Высокая даже на ограниченных ресурсах
- **Простота развертывания**: Один бинарный файл
- **Стабильность**: Без garbage collection пауз

## Ключевые особенности решения

### 1. Оптимизация ресурсов
- Одна задача конвертации за раз
- Минимальное потребление памяти
- Быстрый запуск приложения

### 2. Простота использования
- Веб-интерфейс для управления
- Real-time отслеживание прогресса
- Автоматическое сканирование директорий

### 3. Надежность
- Система очередей с повтором ошибок
- Детальное логирование
- Простое восстановление после сбоев

## Поток работы

1. **Пользователь** добавляет файлы через веб-интерфейс
2. **Backend** сохраняет задачу в SQLite
3. **FFmpeg** конвертирует аудиодорожку
4. **WebSocket** отправляет обновления прогресса
5. **Frontend** отображает статус в реальном времени
